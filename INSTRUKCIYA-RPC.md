# Как получить бесплатный RPC-ключ для BNB Chain (инструкция простым языком)

## Зачем это нужно

Наш сайт сам сканирует блокчейн, чтобы видеть, когда клиенты присылают USDT.
Сейчас он стучится в общую бесплатную «дверь» Binance, а она пускает слишком редко —
поэтому депозиты не находятся. Нужен персональный бесплатный ключ — это как личный
пропуск вместо общей очереди. Денег не стоит, занимает 10 минут.

## Кто это делает

Владелец проекта (вы). Понадобится только адрес электронной почты.

## Где получить (любой из 3 вариантов, достаточно одного)

| Сервис | Сайт | Тариф |
|---|---|---|
| Ankr | https://www.ankr.com | бесплатный |
| NodeReal | https://nodereal.io | бесплатный |
| QuickNode | https://www.quicknode.com | бесплатный |

Рекомендуем Ankr — там проще всего.

## Что сделать, по шагам

1. **Зарегистрируйтесь** на выбранном сайте по email (придёт письмо — подтвердите адрес).
2. **Создайте проект / endpoint.** Кнопка обычно называется «Create Project»,
   «Create Endpoint» или «New API Key».
3. **Выберите сеть:** BNB Smart Chain (BSC), вариант **Mainnet** (не Testnet!).
4. **Скопируйте HTTPS-ссылку** endpoint'а. Она выглядит примерно так:
   - у Ankr: `https://rpc.ankr.com/bsc/КЛЮЧ`
   - у NodeReal: `https://bsc-mainnet.nodereal.io/v1/КЛЮЧ`
   - у QuickNode: `https://имя-проекта.bsc.quiknode.pro/КЛЮЧ/`

   Где `КЛЮЧ` — длинный набор букв и цифр. Нужна вся ссылка целиком.

## Самый простой вариант

**Просто пришлите эту ссылку ассистенту в чат — он сам всё впишет и проверит.**
Дальше ничего делать не нужно.

## Если хотите вписать сами

1. Откройте файл `/var/www/crypto-exchange/backend/.env` и найдите строку `BSC_RPC_URL=`.
   Вставьте туда вашу ссылку:

```
BSC_RPC_URL=https://rpc.ankr.com/bsc/ваш_ключ
```

2. Выполните в терминале сервера:

```bash
cd /var/www/crypto-exchange/backend && php artisan config:cache && systemctl restart crypto-indexer
```

## Как проверить, что заработало

Выполните:

```bash
journalctl -u crypto-indexer -n 20 --no-pager
```

В последних строках **не должно быть** ошибок `limit exceeded`. Если их нет —
сканер депозитов работает, и входящие USDT будут зачисляться на балансы клиентов.

> Важно: ссылка с ключом — это секрет. Не публикуйте её и не отправляйте посторонним.
